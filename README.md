## Description

- In this project, we had to recreate a simple version of wolfenstein3D, to introduce only the basic logic of the game.
- The challenge of this program was to navigate through the minilibx, and create a "3D world" simulated by using as input a .cub file, by
	- Using raycasting : this method has been widely used to create the graphics of the first "First Person Shooter" videogame genre.
		It traces a ray from the player to the different intersection with the grid of the map, allowing to render surroundings textures based on the distance.

## Instructions

- To launch the game, give a valid .cub in argument that respects the criteria by executing this line :
	./cub3D /path/to/map

In case of an invalid map, the game should not begin
